Why WealthFlow was made?

Purpose

Welcome to WealthWise - the platform designed to empower your financial decisions with intelligent tools and insightful data. WealthWise is a front-end web application created to help users visualize and manage their personal finances in one placeâ€”tracking assets, monitoring investment performance, exploring new products, and converting currencies. Built using dummy JSON data and API integrations, it demonstrates how a typical wealth management dashboard functions without requiring a back-end server.

Target Audience and Use Cases

1. Investment Advisors: Use WealthWise to view AUM, manage client portfolios, and recommend investment products.

2. Portfolio Managers: Track multiple portfolios, review ROI, and compare different asset allocations.

Sidebar Menu Features

  • Dashboard: Provides a summary of total assets, revenue, client count, and investment trends. It serves as an overview of key financial metrics.
  • Analytics: Allows users to dive deeper into investment data, generate insights, and track performance metrics through charts and reports.
  • Products: Displays a catalog of financial products like mutual funds, bonds, and ETFs, with options to filter and explore details.
  • Currency Converter: A tool to convert between different currencies using real-time exchange rates, helping users plan international investments.
  • Trending News: Fetches and displays the latest financial news and trends to keep users informed about market developments.
  • About Us: Provides information about the platform's purpose, target audience, and features, helping users understand the application's vision and goals.

High-Level Plan

WealthWise is a small-scale front-end web application that mimics the features of professional wealth management platforms. By using dummy JSON data and a news API, the platform showcases how financial data can be visualized and explored through intuitive interfaces.

In Real-Life Platforms: Similar tools are widely used by investment professionals and advisors to manage portfolios, track performance, and analyze financial trends in real-time.
